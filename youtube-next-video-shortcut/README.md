# YouTube — Next video (N)

Atomic userscript. Press **N** anywhere on a YouTube page to jump to the next video.

Does one thing only.

## Install

[![Install](https://img.shields.io/badge/install-userscript-blue)](https://github.com/oriz-org/userscripts/raw/main/youtube-next-video-shortcut/youtube-next-video-shortcut.user.js)

Requires Tampermonkey, Violentmonkey, or Userscripts (Safari).

## How it works

Clicks `.ytp-next-button` on keydown. Ignores the key when typing in an input/textarea/contenteditable element.
